## pywizlight

WizZ Desktop uses `pywizlight`, an open-source Python library for WiZ device interoperability.

Project:

https://github.com/sbidy/pywizlight

WizZ Desktop uses pywizlight for:

- WiZ device discovery support.
- WiZ protocol compatibility helpers.
- RGB/CW color conversion utilities.
